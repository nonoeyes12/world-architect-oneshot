# Design Philosophy

**The world is a pressure system, not an encyclopedia.** History creates institutions;
institutions create incentives; incentives create factions; factions create conflicts; conflicts
create secrets; secrets create player discovery; discovery changes choices; choices change the
ending. Build along that chain and the setting stays connected and playable.

**Depth must become play.** If there is a myth, someone uses it to justify power. If there is a
taboo, breaking it matters. If there is an ancient disaster, its consequences shape geography, law,
religion, or daily life. If there is a monster, it means something beyond a stat block.

**Entertainment is engineered rhythm.** The best one-shots alternate pressure and release, clarity
and mystery, danger and wonder, comedy and dread, intimacy and spectacle. Players remember the
priest who protected the monster because the monster was the town's only clean water; the market
where every song hid a legal contract; the choice between saving the village and preserving the
truth.

**Agency must be real.** No scripted sequence. Build nodes, clocks, motives, and consequences, and
give players several ways to reach the truth, several ways to confront the conflict, and several
possible endings.

**The GM needs handles.** Beautiful prose isn't enough. The GM needs names, motives, sensory
details, clocks, fallback clues, shortened and expanded scenes, improv tables, and clear stakes —
tools to run the game under time pressure. The Run Sheet exists so the whole thing can be run
without re-reading the document.
